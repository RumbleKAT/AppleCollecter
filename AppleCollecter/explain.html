홈 해외 ios 기사 모음 즐겨찾기 제작방법

사용 기술 설명

tag, css, jQuery(js 포함)에 대한 설명 페이지입니다.

태그 작성

반복해서 사용되는 것은 class 생성자를, 특정 요소를 표현 하기 위해서 id 생성자를 이용했습니다.
특히 div에 id를 두어서 section -> article -> content로 이어지는 태그의 흐름을 제작했습니다.

css 스타일 지정

자식 선택자와 후손 선택자를 활용하여 큰 태그부터 작은 세부적인 태그로 스타일을 적용시켜나갔습니다.










미디어 태그의 활용

미디어 태그를 활용하여 스마트폰과 태블릿 그리고 PC에 알맞는 이미지 스타일을 적용했습니다.
특히 이미지의 경우 화면에 따라 비율이 달라지기 때문에 더욱 신경써서 제작했습니다.
각 플랫폼 마다 사이즈를 확인하여 padding이나 width의 비율을 재 정의 했습니다.

AJAX의 활용

비동기 통신인 AJAX를 이용하여 NEWS JSON 정보를 수신했습니다.
JSON 파일의 element들을 연결하여 콜백함수를 이용하여 그 값을 처리했습니다.
클라우드에 올려 놓은 자체 서버와 직접 연결하여 번역 기능을 만들었습니다.

AJAX의 활용

먼저 해당 언어가 무엇인지 Request를 보내고 번역할 수 있는 언어라면 다시 한번 Request를 보내어 번역이 가능하도록 만들었습니다.
만약 번역이 불가능한 언어라면 alert 메서드를 이용해 불가능 메시지를 띄웠습니다.

LocalStorage의 활용

즐겨찾기에 저장한 데이터를 myDB라는 key값에 넣어서 localStorage 객체에 저장했습니다.
저장한 객체 값은 다른 페이지에서도 사용 가능한 쿠키 데이터가 됩니다.
이를 통해 즐겨찾기 목록에서 위의 데이터를 수정할 수 있도록 기능을 추가했습니다.

JQuery의 활용

hover와 animate 그리고 attr등 다양한 Jquery 문법을 활용하여 제작했습니다.
이미지를 선택하면 다른 이미지로 바뀌고 해당 이미지를 다시 선택하면 원래의 이미지로 돌아가는 토글 기능을 만들었습니다.
마우스가 위에 있으면 해당 이미지 주위에 초록색 사각형이 생기도록 제작했습니다.
처음 화면을 로드하면 서서히 div가 줄어들수 있도록 제작했습니다.
